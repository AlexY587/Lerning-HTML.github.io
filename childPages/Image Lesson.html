Steps in building an image
  • Pick an image (be cautious of the file extension in the URL)
  • Record URL, cite it, comment for the date accessed, and record the original dimensions
  • Use blockquote and comment as approiate
  • Use img-tag with atributes scr, alt, title. and style with width-height
  • Note: openign in a new tab will eventually provide a http or https URL with a file extension
